# DBS Project — Protein Store

A database-driven protein store management system built for DBMS coursework.

## Features

* Product listing
* Order management
* Admin dashboard
* Order history
* Sales statistics

## Tech Stack

* MySQL
* Node.js / Express (if applicable)
* HTML/CSS/JS

## How to Run

1. Clone repository
2. Import schema.sql
3. Run backend
4. Open frontend
