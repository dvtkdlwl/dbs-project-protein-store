# DBS Lab Final Project: Protein Store

A database-driven protein store management system built as a final project for my Database Systems lab.

## Features

* Product listing
* Order management
* Admin dashboard
* Order history
* Sales statistics

## Tech Stack

* MySQL
* Node.js / Express (if applicable)
* HTML/CSS/JS
